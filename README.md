# GlowFace AI

Simple React + Vite app for running the glow face feature component at `src/features/glow-face/GlowFaceApp.tsx`.

## Run locally

```bash
pnpm install
pnpm dev
```

Then open the local URL shown in terminal (usually `http://localhost:5173`) and allow camera access in your browser.

## Build

```bash
pnpm build
pnpm preview
```
